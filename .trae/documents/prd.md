## 1. 产品概述

民宿房源管理系统，面向民宿运营方，提供多套房源的分区展示、日历可订视图、预约时段互斥校验、入住结算、退改违约金核算、入住档案生成及后台营收统计等功能，覆盖民宿行业从房源管理到入住结算的全流程业务。

## 2. 核心功能

### 2.1 用户角色
| 角色 | 注册方式 | 核心权限 |
|------|----------|----------|
| 管理员 | 系统预设 | 房源管理、订单管理、保洁安排、营收统计、系统配置 |
| 前台/客服 | 管理员分配 | 预约下单、订单查看、入住办理、退改处理 |
| 住户 | 无需注册 | 浏览房源、查看可订日期、提交预约 |

### 2.2 功能模块
1. **房源总览页**：分区展示多套民宿，标注户型、配套、价格，日历视图展示可订日期
2. **预约下单页**：选择房源与时段，自动跨天时段互斥校验，选配入住人数与额外服务
3. **订单管理页**：订单列表与详情，支持在线退改，按规则核算违约金
4. **入住档案页**：入住后生成入住档案，记录入住信息与结算明细
5. **后台管理页**：房源状态管理、保洁安排、营收统计仪表盘

### 2.3 页面详情
| 页面名称 | 模块名称 | 功能描述 |
|----------|----------|----------|
| 房源总览页 | 房源分区卡片 | 按区域分组展示房源，每张卡片标注户型（整租/单间）、配套设施（WiFi/厨房/空调等）、基础价格 |
| 房源总览页 | 日历可订视图 | 月度日历视图，绿色标记可订日期，红色标记已订日期，灰色标记维护中日期，点击日期查看当日详情 |
| 房源总览页 | 房源筛选器 | 按区域、户型、价格区间、配套设施筛选房源 |
| 预约下单页 | 时段选择器 | 选择入住日期与离店日期，自动计算入住天数 |
| 预约下单页 | 时段互斥校验 | 提交前自动校验所选时段是否与已有订单重叠，若重叠则提示冲突并阻止提交 |
| 预约下单页 | 人数与服务选配 | 选择入住人数（超出基础人数加收费用），勾选额外服务（早餐/接机/洗衣等） |
| 预约下单页 | 费用预览 | 实时计算房费+加人费+额外服务费，展示费用明细 |
| 订单管理页 | 订单列表 | 按状态（待入住/入住中/已完成/已取消）筛选订单，支持搜索 |
| 订单管理页 | 退改操作 | 在线发起退改申请，系统按退改规则自动核算违约金，确认后执行退改 |
| 订单管理页 | 订单详情 | 展示订单完整信息：房源、时段、费用、退改记录、入住档案 |
| 入住档案页 | 档案生成 | 入住后自动生成档案，记录住户信息、入住时段、实际费用 |
| 入住档案页 | 档案列表 | 按时间倒序展示所有入住档案，支持搜索与导出 |
| 后台管理页 | 房源状态管理 | 切换房源状态（可订/已订/维护中），设置不可订日期段 |
| 后台管理页 | 保洁安排 | 为离店后的房源安排保洁任务，标记保洁状态（待保洁/保洁中/已完成） |
| 后台管理页 | 营收统计 | 仪表盘展示：总营收、入住率、平均房价、热门房源排名，支持按月/季/年筛选 |

## 3. 核心流程

### 3.1 预约下单流程
用户浏览房源 → 选择房源查看日历可订日期 → 选择入住/离店日期 → 系统校验时段互斥 → 选配人数与额外服务 → 确认费用明细 → 提交订单 → 订单生成

```mermaid
flowchart TD
    "浏览房源列表" --> "选择房源"
    "选择房源" --> "查看日历可订日期"
    "查看日历可订日期" --> "选择入住/离店日期"
    "选择入住/离店日期" --> "时段互斥校验"
    "时段互斥校验" -->|"通过"| "选配人数与额外服务"
    "时段互斥校验" -->|"冲突"| "提示冲突信息"
    "提示冲突信息" --> "重新选择日期"
    "重新选择日期" --> "时段互斥校验"
    "选配人数与额外服务" --> "确认费用明细"
    "确认费用明细" --> "提交订单"
    "提交订单 --> 订单生成"
```

### 3.2 退改流程
用户查看订单 → 发起退改申请 → 系统按规则核算违约金 → 展示退改费用明细 → 确认退改 → 订单状态变更 → 释放房源时段

```mermaid
flowchart TD
    "查看订单" --> "发起退改申请"
    "发起退改申请" --> "核算违约金"
    "核算违约金" --> "展示退改费用明细"
    "展示退改费用明细" --> "确认退改"
    "确认退改" --> "订单状态变更"
    "订单状态变更" --> "释放房源时段"
```

## 4. 用户界面设计

### 4.1 设计风格
- 主色调：温暖大地色系（#8B6F47 棕色 / #D4A574 驼色），传递民宿的温馨与自然感
- 辅助色：#2D5016 森林绿（可订状态）、#C41E3A 深红（已订/警示）、#F5F0EB 米白（背景）
- 按钮风格：圆角（8px），主操作按钮带微弱阴影，hover 时阴影加深
- 字体：Noto Serif SC（标题）+ Noto Sans SC（正文），标题 24px/20px，正文 14px/12px
- 布局风格：左侧导航栏 + 右侧内容区，卡片式布局，顶部面包屑导航
- 图标风格：线性图标（lucide-react），配合圆角方形背景色块

### 4.2 页面设计概览
| 页面名称 | 模块名称 | UI 元素 |
|----------|----------|---------|
| 房源总览页 | 房源分区卡片 | 网格布局，每张卡片含房源图片、户型标签、配套图标组、价格标注，hover 时微上浮 + 阴影增强 |
| 房源总览页 | 日历可订视图 | 月度日历网格，日期格带色点标记（绿/红/灰），点击日期弹出当日订单摘要浮层 |
| 房源总览页 | 房源筛选器 | 顶部横向筛选条，下拉选择 + 标签式多选，毛玻璃背景 |
| 预约下单页 | 时段选择器 | 双月日历并排，拖选日期范围，选中区域高亮，冲突日期标红 |
| 预约下单页 | 人数与服务选配 | 步进器控件调人数，复选框列表选服务，右侧固定费用预览面板 |
| 预约下单页 | 费用预览 | 固定侧栏，逐行列出房费/加人费/服务费/总计，底部确认按钮 |
| 订单管理页 | 订单列表 | 表格布局，状态标签带色条，行展开显示详情 |
| 订单管理页 | 退改弹窗 | 居中弹窗，显示退改规则与违约金计算过程，确认/取消双按钮 |
| 入住档案页 | 档案卡片 | 时间轴布局，每条档案为信息卡片，含住户信息与费用明细 |
| 后台管理页 | 营收统计仪表盘 | 顶部统计卡片（总营收/入住率/均房价），中部折线图 + 柱状图，底部热门房源排行 |

### 4.3 响应式设计
- 桌面优先（1280px+），适配平板（768px-1279px）和手机（< 768px）
- 桌面端：左侧导航 + 右侧内容，双月日历并排
- 平板端：顶部导航 + 单列内容，单月日历
- 手机端：底部导航 + 单列内容，日历简化视图
